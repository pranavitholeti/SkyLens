# 🌤 SkyLens - Real-Time Weather Forecast Application

SkyLens is a modern and responsive weather forecasting web application built using **HTML**, **CSS**, and **Vanilla JavaScript**.
The application fetches live weather data from the **OpenWeatherMap API** and dynamically displays weather information for any searched city.

---

# 🚀 Features

## ✅ Current Weather Information

* City Name
* Temperature
* Weather Condition
* Weather Icon
* Humidity
* Wind Speed
* Feels Like Temperature
* Current Date

---

## ✅ 5-Day Weather Forecast

* Displays upcoming weather forecast
* Shows temperature and weather condition using icons

---

## ✅ Additional Functionalities

* Search weather by city
* Current location weather using Geolocation API
* Temperature conversion (°C ↔ °F)
* Dark/Light theme toggle
* Recent searches using Local Storage
* Dynamic weather backgrounds
* Loading spinner while fetching data
* Error handling for invalid cities and network issues
* Fully responsive design
